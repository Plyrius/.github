# Plyrius

**Open data tools, built in public.**

Plyrius is an open-source lab that researches and builds tools around public data —
free APIs, visual interfaces, and curated datasets that make structured information accessible to developers, researchers, and students.

> The name derives from *pliro* (πληρο), Greek for data and information.

## Philosophy

- **Free tools.** APIs and UIs are free and open source.
- **Paid data.** Datasets are paid to keep the work sustainable. Researchers and students can reach out for access.
- **Quality over quantity.** Everything is documented, sourced, and maintained.
